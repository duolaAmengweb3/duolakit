USE IT
───────────────────────────────────────────────────────────────

  /openapi-check          # detect drift (read-only)
  /openapi-sync           # auto-fix (asks before each write)
  /openapi-init           # bootstrap a new Express + OpenAPI project

Try the demo (clone the repo, then):
  cd duolakit/01-openapi-guardian/examples/express-demo
  npm install
  npm run dev

Then in Claude Code, /openapi-check will find the intentional
DELETE /users/{id} drift. /openapi-sync will fix it.

A Fastify variant is at examples/fastify-demo/ — same drift,
proves the Pro framework support works.
