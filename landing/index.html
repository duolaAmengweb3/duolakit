What "production-grade" means here

Out of 9,000+ Claude Code plugins, ~100 are production-ready. duolakit aims to be in that 100.

  • Unit tests + a 30-second demo video, every plugin
  • Bilingual README (中文 + English)
  • Free MIT core forever — no rug-pull to paid
  • Clear Pro license boundary, separately listed
  • Monthly release cadence; issues answered within 30 days
  • One author, one brand — accountability not committee
